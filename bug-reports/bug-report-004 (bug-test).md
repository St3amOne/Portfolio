## Краткое описание
Форма отправляется без обязательного подтверждения согласия на обработку персональных данных (чекбокс не активен).

## Шаги воспроизведения
1.  Открыть в браузере страницу `https://bug-test.online/`.
2.  Заполнить обязательные текстовые поля валидными данными.
6.  Нажать кнопку "Отправить".

## Фактический результат
После нажатия кнопки "Отправить" данные уходят на сервер, появляется уведомление "Форма отправлена"

## Ожидаемый результат
Должно появляться предупреждение "Необходимо дать согласие на обработку данных", пока пользователь не отметит чекбокс о персональных данных.
