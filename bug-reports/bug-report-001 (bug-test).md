## Краткое описание
При нажатии на ссылку **«задание»** в навигационной панели происходит некорректный якорный переход к блоку **«типы багов»**, вместо перехода к отсутствующему блоку с заданием.

## Шаги воспроизведения
1.  Открыть в браузере страницу `https://bug-test.online/`.
2.  В верхнем навигационном меню найти пункт **«задание»**.
3.  Нажать левой кнопкой мыши на ссылку **«задание»**.

## Фактический результат
Страница прокручивается к разделу **«типы багов»**.

## Ожидаемый результат
Страница прокручивается к разделу, который описывает **задание** для тестировщика.

## Приоритет
**Medium**. Баг заметен, мешает навигации, но не блокирует использование основного функционала сайта.

## Серьёзность
**Minor**. Ошибка не связана с критической функцией.

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11
- **Браузер:** Edge 145.0.3800.58
- **Разрешение экрана:** 1920x1080
